Accessing different Mission-ID's

Our system defaults to Mission-ID 1.
If you want to access the latest Data or historical Data you can simply add a "?id=x" to the URL, whereas x is the ID you want to access.
For example: "&id=2" will display the latest Data or historical Data from Mission 2.

Downloading Data

If you want to download our Data there is a way to display it as a .json.
To do so add a "?mode=json" to the URL.
To change the Mssion-ID and display the Data as .json you can combine both methods, for example by adding "?id=2&mode=json".
This will display the Data of mission 2 as .json
